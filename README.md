# BOM智能提取助手

> 小鹏AI公开赛 赛题三：效能跃升·AI开挂  
> 专为机械设计制造领域打造的BOM（物料清单）智能提取工具

## 痛点

机械工程师每月需处理大量BOM表：从纸质图纸、PDF装配图、Excel明细表中逐行录入零件号、名称、材料、规格、数量等信息。一张复杂BOM表（50+项）手动录入耗时30-60分钟，且容易出错。

## 解决方案

**上传BOM → AI识别 → 自动校验 → 一键导出ERP格式**

| 功能 | 说明 |
|------|------|
| 多格式输入 | 图片（拍照/截图）、PDF、Excel |
| AI视觉识别 | 通义千问VL自动识别表格内容 |
| 图像预处理 | 自动纠偏、对比度增强、锐化 |
| 材料标准化 | 45钢→45#，304→06Cr19Ni10，20+种映射 |
| 来源自动分类 | 自制件/外购件/标准件/外协件/焊接件/铸造件 |
| 数据校验Agent | 自动检查重量一致性、材料牌号、零件号重复、数量有效性 |
| 在线编辑审核 | 可直接增删改，人工复核 |
| ERP导出 | 标准BOM Excel + 用友/金蝶导入模板 |
| 飞书推送 | 提取完成自动推送通知到飞书群 |
| 效率报告 | 自动生成时间节省对比数据 |

## 快速启动

### 环境要求

- Python 3.10+
- Windows / macOS / Linux

### 安装

```bash
cd bom-assistant
pip install -r requirements.txt
```

### 配置

复制 `.env.example` 为 `.env`，填入API Key：

```bash
DASHSCOPE_API_KEY=sk-你的key
DASHSCOPE_BASE_URL=https://dashscope.aliyuncs.com/compatible-mode/v1

# 可选：飞书推送
FEISHU_WEBHOOK_URL=https://open.feishu.cn/open-apis/bot/v2/hook/...
FEISHU_WEBHOOK_SECRET=你的签名密钥
```

### 启动

```bash
streamlit run app.py
```

或双击 `run.bat`（Windows）。

浏览器访问 `http://localhost:8502`

## 使用流程

1. **上传BOM文件** — 选择图片/PDF/Excel，支持批量上传
2. **AI智能提取** — 点击按钮，AI自动识别并结构化
3. **数据校验** — 运行自动校验，检查数据一致性
4. **审核编辑** — 在表格中修正错误
5. **导出** — 下载BOM Excel / ERP模板 / 效率报告
6. **飞书推送** — 一键发送通知到飞书群

## 技术架构

```
用户上传文件
    ↓
图像预处理（OpenCV纠偏+CLAHE+锐化）
    ↓
┌─────────────────────────────────┐
│  Excel文件 → pandas直接解析     │
│  图片/PDF → 通义千问VL视觉识别  │
└─────────────────────────────────┘
    ↓
BOM提取Agent（JSON结构化+材料标准化+来源分类）
    ↓
数据校验Agent（重量一致性/材料牌号/重复检查/数量校验）
    ↓
人工审核（在线编辑表格）
    ↓
导出Agent（BOM Excel + ERP模板 + 效率报告）
    ↓
飞书推送Agent（通知卡片）
```
